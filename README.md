# Sistema Financiero BEB

## Descripción
Sistema bancario desarrollado en Java (NetBeans) con MySQL y JDBC, que permite gestionar clientes, cuentas y operaciones como consignaciones, retiros y CDT.

## Funcionalidades
- Registro de clientes
- Gestión de cuentas bancarias
- Consignaciones y retiros
- Apertura y cierre de CDT
- Generación de reportes

## Tecnologías utilizadas
- Java
- NetBeans
- MySQL
- JDBC
